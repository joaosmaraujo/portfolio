Infraspeak Dashboard

Main Panel

In the main panel the Infraspeak team has access to some general stats about Infraspeak usage. This stats are successively updated so the info is always up-to-date. As well as in the other UI panels, the users can access the other options through the side menu.

View Builder

The View Builder allows users to setup and save new views by chosing which metrics should be placed in each panel block. The user selects for each block some parameters and one of the analysis options available. The results are shown in each block in a clear and easy to read presentation.

Standard View

This screenshot shows one of the saved views by the dashboard users. The views built using the View Builder can then be accessed through the Saved Views menu option. The user may apply some filters to the saved views which will change the results and obviously the data presentation.
