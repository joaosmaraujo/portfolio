Infraspeak Dashboard
Schedule: March 2016 to June 2016

The last 3 months I'd been developing a web app which is an internal tool to Infraspeak. This app is a dashboard that allows Infraspeak team to evaluate how their customers use their SaaS. My job was to understand what were the most important features required by the Infraspeak team and then to design the system as well as the database and the UI.


Languages and Tecnhologies

  • Frontend: HTML5, CSS3, Bootstrap, Javascript, jQuery
  • Backend: PHP, Laraval 5.2 framework
  • Database: PostgreSQL
  • Other Relevant Stuff: jQuery AJAX and some Javascript data presentation libraries such as ChartJS and ECharts
